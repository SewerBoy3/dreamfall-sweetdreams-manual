# Dreamfall: Sweetdreams - Manual de Supervivencia

Este repositorio contiene el manual de instrucciones en formato fanzine de **Dreamfall: Sweetdreams**, un videojuego Metroidvania 2D (actualmente en pausa) desarrollado en Godot 4. 

El fanzine fue creado usando HTML y CSS puro para participar en la **ManualJam**.

## Sobre el Juego

**Dreamfall: Sweetdreams** sigue la historia de Alex, un joven consumido por la rutina y el aislamiento. Su única vía de escape son los sueños lúcidos. Sin embargo, su subconsciente ha transformado sus traumas y miedos en siete reinos independientes.

A lo largo del juego, Alex debe explorar su propia mente, desbloquear habilidades de movimiento (como el *Dash*, el *Doble Salto* y la *Gravedad Inversa*) y derrotar a las manifestaciones de sus miedos, culminando en un enfrentamiento final contra "El Alex del Pasado" para alcanzar la Trascendencia.

## Cómo ver el manual

El manual está diseñado en tamaño A5, con una estética melancólica que mezcla arte pixelado (Kings and Pigs) y notas manuscritas. 

Tienes dos formas de leerlo:
1. **[Recomendado] PDF:** Descarga el archivo `Dreamfall_Manual.pdf` desde la sección de *Releases* o directamente desde los archivos del repositorio.
2. **Versión Web:** Abre el archivo `manual.html` en cualquier navegador web. (Nota: Para exportarlo a PDF desde el navegador, usa `Ctrl + P`, selecciona tamaño A5 y asegúrate de activar "Gráficos de fondo").
